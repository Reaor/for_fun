# 字宙 ZIVERSE · 汉字观测终端

> 一台来自"平行历 1984"的复古未来主义终端机。
> 它捕获了汉字的电子幽灵 —— 它们仍然活着，并且可以被触摸。

纯前端、零依赖、零构建。直接用浏览器打开 `index.html` 即可接入字宙。

## 四个实验舱

| 舱位 | 名称 | 玩法 |
|------|------|------|
| 01 | **粒子铸字** GLYPH FOUNDRY | 输入任意汉字，将其熔成数千枚光粒子；用指针搅动它，字会散开、又会自己回家。支持霓虹 / 磷光 / 琥珀 / 余烬四种光谱与"引爆"。 |
| 02 | **字源回溯** TIME MACHINE | 拖动时间轴，看「日月山水火木」从甲骨文刻痕渐变为屏幕上的今体字（字形为手绘示意重构），时间读数从公元前 1300 年滚动到今天。 |
| 03 | **拆字炼金所** GLYPH ALCHEMY | 汉字是唯一能做加法的文字：日＋月＝明，木＋木＝林，林＋木＝森……共 16 道隐藏配方，产物可链式再合成。 |
| 04 | **诗境雨幕** POEM RAIN | 数据雨里混进了一句唐诗。在字符坠落消失之前点中它们，把诗一个字一个字打捞回来；每接住一字，奏响一枚五声音阶。 |

## 细节彩蛋

- 开机有完整的 BIOS 引导序列（按任意键接入）
- 顶栏时钟运行在"平行历"（公历 +93 年）
- 连点主视觉「字宙」五次，触发故障风暴
- 打开浏览器控制台，有一句话留给你
- 所有音效由 WebAudio 实时合成，宫商角徵羽五声音阶

## 技术

原生 HTML / CSS / JavaScript。Canvas 粒子物理、SVG 字形手绘、IntersectionObserver、WebAudio 合成器。无任何外部依赖（字体来自 Google Fonts，离线时优雅回退到系统字体）。
